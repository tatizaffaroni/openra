# openra

Is a project that recreates and modernizes the classic Command & Conquer real time strategy games.

URL: http://www.openra.net/

Needed from KCP:
[qarma](../../../qarma)

Install in this order:
```
kcp -di qarma 
```
```
kcp -i openra
```
